Detailed Description

D/A controller SPI class definition.

We interface to the LTC2754 chip through a standard SPI interface defined in AlteraSpi.

Member Enumeration Documentation

◆ ChannelNbr

enum Ltc2754_SPI::ChannelNbr : uint8_t

Definition of channel number possible parameters.

The commands to the LTC2754 have typically as an option the channel number, or all the channels, or if we want to disable part of a command, no channel.

Enumerator
CHANNEL_0 

Channel #0, first channel.

CHANNEL_1 

Channel #1, second channel.

CHANNEL_2 

Channel #2, third channel.

CHANNEL_3 

Channel #3, fourth channel.

CHANNEL_ALL 

All the channels are selected.

CHANNEL_NONE 

No channel, option disabled.
